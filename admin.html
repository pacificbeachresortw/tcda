JSONBin.io 設定
  1. 前往 jsonbin.io 免費註冊。
  2. 建立新 Bin，初始內容：{"news":[]}，按下 Create。
  3. 從 URL 複製 Bin ID（如 6abc123def456）。
  4. 右上角頭像 → API Keys，複製 X-Master-Key。
ImgBB 圖片上傳設定
  1. 前往 imgbb.com 免費註冊。
  2. 進入 api.imgbb.com，點選 Get API key。
  3. 複製 API Key 貼入下方。
